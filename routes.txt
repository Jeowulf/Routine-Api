/routines
GET - gets all routines for a user
POST - creates a routine
  body = {
          routineName: String,
          routineStyle: String, // '4-day',  'complex'
          status: Number,
          completed: Boolean,
          startDate: Number,
          endDate: Number,
          createdDate: Number,
          modifiedDate: Number,
          userId: String,
        }

/routines/routine_id
GET - gets single routine
PUT - edits routine
  body is same as post
DELETE - deletes routine
